# PeiliangCai.github.io

欢迎来到我的个人博客！

这个项目是基于 **Vue 3 + Vite** 构建的一个现代化、动态感的个人博客，专门为展示学术科研成果和个人技术心得而设计。

## 🚀 特性
- 💎 **现代视觉设计**: 采用了 HSL 色彩系统和玻璃拟态效果。
- 🌓 **深色模式**: 支持一键切换亮色/暗色主题。
- 📋 **科研专区**: 特化的科研与项目展示模块。
- 🤖 **自动化部署**: 使用 GitHub Actions 实现代码推送即自动上线。

## 🛠️ 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📦 部署指引
当代码推送到 `main` 分支时，GitHub Actions 会自动进行构建并部署。

确保在 GitHub 仓库设置中：
1. `Settings` -> `Pages`
2. `Build and deployment` -> `Source` 选择 **GitHub Actions**。
